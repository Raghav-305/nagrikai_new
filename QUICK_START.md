# ⚡ Quick Start Guide - Nagrik AI

Get the complete Nagrik AI system running in 5 minutes!

## Prerequisites Checklist

- [ ] Node.js 16+ installed
- [ ] Python 3.13+ installed  
- [ ] Docker & Docker Compose installed (optional, for containerized setup)
- [ ] Groq API key (sign up at https://console.groq.com)
- [ ] Text editor or IDE (VS Code recommended)

## Option 1: Docker Compose (Fastest ⭐ Recommended)

**Estimated Time: 5 minutes**

### Step 1: Configure Environment
```bash
cd Nagrik_AI

# Set your Groq API key
# Edit ai_server/.env:
GROQ_API_KEY=your-actual-groq-api-key-here

# Edit server/.env:
AI_SERVER_TOKEN=your-secret-token-here
```

### Step 2: Start All Services
```bash
docker-compose up -d
```

### Step 3: Wait for Services to Start
```bash
# Watch the logs
docker-compose logs -f

# All services should show "healthy" or "running"
```

### Step 4: Access the Application
- **Frontend**: http://localhost:3000
- **Backend API**: http://localhost:5000
- **AI Server**: http://localhost:7860

✅ **Done!** Start submitting complaints!

## Option 2: Manual Setup (Detailed Control)

**Estimated Time: 15 minutes**

### Terminal 1: Backend Server

```bash
cd server

# Copy and configure environment
cp .env.example .env
# Edit .env with your settings

# Install and start
npm install
npm run dev

# Should output: "Server running on port 5000"
```

### Terminal 2: Python AI Server

```bash
cd ai_server

# Create Python environment
python -m venv venv

# Activate environment
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Copy and configure environment
cp .env.example .env
# Edit .env with Groq API key

# Install and start
pip install -r requirements.txt
python -m uvicorn server:app --reload --port 7860

# Should output: "Application startup complete"
```

### Terminal 3: Frontend

```bash
cd client

npm install
npm run dev

# Should output: "VITE v... ready in ... ms"
# Frontend will open at http://localhost:3000
```

✅ **All services running!**

## What's Happening Behind the Scenes?

When you submit a complaint:

1. **Frontend** → Sends text + image + location to Backend API
2. **Backend** → Validates and calls AI Server
3. **AI Server** → Multi-agent system analyzes:
   - ORCHESTRATOR routes to right department
   - Specialist agent analyzes in detail
   - AUDITOR verifies assignment
4. **Result** → Ticket created with AI insights

## Common Issues & Quick Fixes

### ❌ "Cannot connect to AI Server"
```bash
# Check if AI server is running
curl http://localhost:7860/health

# If not, restart it:
docker-compose restart ai-server
# Or in terminal: Ctrl+C and restart the command
```

### ❌ "Unauthorized - invalid token"
**Fix**: Make sure tokens match in both .env files
```bash
# server/.env
AI_SERVER_TOKEN=your-token

# ai_server/.env  
INTERNAL_SECRET_TOKEN=your-token  # ← Must be SAME
```

### ❌ "No module named langchain_groq"
**Fix**: Install dependencies again
```bash
cd ai_server
pip install -r requirements.txt
```

### ❌ "Port 5000/3000/7860 already in use"
**Fix**: Use different ports
```bash
# Backend
npm run dev -- --port 5001

# Frontend
npm run dev -- --port 3001

# Update .env files accordingly
```

## Useful Commands

### Docker Compose

```bash
# Start services
docker-compose up -d

# Stop services
docker-compose down

# Restart specific service
docker-compose restart ai-server

# View logs
docker-compose logs -f

# Rebuild images
docker-compose up --build

# Remove everything (including data!)
docker-compose down -v
```

### Manual Development

```bash
# Backend with debug
NODE_DEBUG=* npm run dev

# Python with verbose logging
python -m uvicorn server:app --reload --log-level debug

# Frontend with source maps
npm run dev
```

### Database

```bash
# Connect to MongoDB
mongosh mongodb://admin:password@localhost:27017/nagrik

# See all databases
show databases

# Use nagrik database
use nagrik

# See all collections
show collections

# View complaints
db.complaints.find().pretty()

# Count complaints
db.complaints.countDocuments()

# Find specific complaint
db.complaints.findOne({ticket_id: "INFR-abc12345"})
```
